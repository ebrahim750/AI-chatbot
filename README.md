# AI-chatbot
Dynamically configured based on client, uses N8N as backend.

## NPM commands
- `npm install` — install dependencies
- `npm run build` — build Tailwind and refresh local assets
- `npm run build:watch` — watch Tailwind during development
- `npm run build:assets` — refresh local Showdown + fonts
- `npm run build:plugin` — copy built assets into the WP plugin folder
